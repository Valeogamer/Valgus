## Valgus
___
#### Плоско-вальгусная деформация стопы

### Цель
___

### **Задачи**
___
- [ ] Изучить метод определения плоско-вальгусной деформации стопы.
- [ ] Рассмотреть полезные библиотеки.
- [ ] Разработать варианты решения.
